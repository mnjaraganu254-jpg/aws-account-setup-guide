Step by Step Process

Part 1 - Create Account
Go to aws.amazon.com
Create account with business email
Verify phone number
Select free tier support plan

Part 2 - Secure Root Account
Enable MFA on root account
Delete root access keys
Never use root for daily work

Part 3 - Create IAM User
Go to IAM service
Create admin user
Attach AdministratorAccess policy
Enable MFA on IAM user

Part 4 - Password Policy
Minimum 12 characters
Require uppercase and lowercase
Require numbers and symbols
Password expires every 90 days

Part 5 - Billing Protection
Enable billing alerts
Create $10 threshold alarm
Create $50 threshold alarm
Configure monthly budget
Enable Cost Explorer
